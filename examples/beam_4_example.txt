# ============================================================================
# ANALYTICAL SOLUTION (for verification):
# ============================================================================
#
# Case 1: Node 4 LIFTS OFF (reaction would be upward, but support is Y- only)
# -------------------------------------------------------------------------
# If Node 4 lifts off, the beam is a cantilever from Node 1 with:
#   - Pin support at Node 2 (100")
#   - Upward load P at Node 3 (150")
#
# For small P, the reaction at Node 2 will be downward, and at Node 4 would
# be upward (tension). Since Node 4 can only push DOWN (Y-), it lifts off.
#
# Statically determinate system (cantilever with one intermediate pin):
#   Sum Fy = 0:  R1y + R2y + P = 0
#   Sum M@1 = 0: R2y(100) + P(150) + M1 = 0
#   Sum M@2 = 0: -R1y(100) + P(50) - M1 = 0
#
# For P = 10 kips upward:
#   From M@2: M1 = R1y(100) + P(50) = R1y(100) + 500
#   From M@1: M1 = -R2y(100) - 1500
#   Therefore: R1y(100) + 500 = -R2y(100) - 1500
#              R1y = -R2y - 20
#   From Fy:   R1y + R2y = -10
#   Substitute: -R2y - 20 + R2y = -10
#              -20 = -10  (ERROR - system is indeterminate!)
#
# Actually, with the pin at Node 2, we need to use moment equilibrium:
#   Sum M@2 = 0: -R1y(100) + P(50) - M1 = 0
#   For cantilever from 1 to 2: M1 = R1y(100) - P(50)
#
# Let me recalculate properly:
# With Node 4 lifted off, we have a propped cantilever (fixed at 1, pin at 2):
#   - Fixed support at Node 1 (moment M1, reactions R1x, R1y)
#   - Pin support at Node 2 (reactions R2x, R2y)
#   - Load P = 10 kips upward at Node 3 (x = 150")
#
# Statics:
#   R1x = 0 (no horizontal loads)
#   R1y + R2y + 10 = 0  →  R1y + R2y = -10
#   M1 + R2y(100) + 10(150) = 0  →  M1 + 100*R2y + 1500 = 0
#
# Compatibility (slope at Node 2 must be continuous):
# For propped cantilever with end load:
#   Using slope-deflection or conjugate beam method...
#   
# Standard result for propped cantilever with load at distance 'a' from fixed end:
#   If L = 100" (fixed to pin), a = 150" is BEYOND the pin!
#
# So the load is actually in the unsupported span (150" to 200").
# The beam from Node 2 to Node 3 acts as a simple cantilever from Node 2.
#
# Simpler approach - just check reaction at Node 4:
#   If Node 4 were active, with 3 supports (1 fixed, 2 pinned), we'd have:
#   Sum M@2 = 0: -R1y(100) + P(50) - M1 + R4y(100) = 0
#
# For this problem, let's use the EPFRAME results and compare!
# The key check is: Does Node 4 lift off or stay in contact?
#
# ============================================================================
# EXPECTED RESULTS:
# ============================================================================
#
# With P = 10 kips UPWARD at Node 3:
#
# IF Node 4 LIFTS OFF (Y- reaction would be upward/positive):
#   - Displacement at Node 4: Positive (upward)
#   - Reaction at Node 4: Zero
#   - The beam acts as cantilever from Node 1 with pin at Node 2
#   - Reaction at Node 2 should be downward (negative)
#
# IF Node 4 STAYS IN CONTACT (Y- reaction is downward/negative):
#   - Displacement at Node 4: Zero
#   - Reaction at Node 4: Negative (downward/compression into support)
#   - Three-support continuous beam
#
# For this specific load (10 kips upward at mid-span between 2 and 4),
# we expect Node 4 to LIFT OFF because the upward load will try to
# create an upward reaction there, but the Y- support can only resist
# downward motion.
